"Équité : Qualité consistant à attribuer à chacun ce qui lui est dû par référence aux principes de la justice naturelle ; impartialité"
Le larousse

Ce projet met à disposition de toutes et tous dans un format facile à exploiter et facile à vérifier les données du CSA concernant l'équité de temps de parole des candidats sur les chaînes de télé et radios.

Les formats disponibles sont des fichiers csv ainsi qu'une API en JSON pour que les développeuses & développeurs puissent créer en toute simplicité et rapidement des graphiques pour visualiser ces données.

Sources :

  • JSON : temps d'antenne du 1er Février au 26 Février sur les chaînes généralistes
  • Fichiers originaux du CSA
  • Code source générant les données mises à disposition par ce site
