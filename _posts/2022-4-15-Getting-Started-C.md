<p data-sourcepos="49:1-49:25" dir="auto"><strong>What is <code>&lt;stdio.h&gt;</code> ?</strong></p>
<p data-sourcepos="51:1-51:215" dir="auto">It stands for 'Standard Input Output'. stdio.h is a header file which has the necessary information to include the input/output related functions in our program. Functions like <code>printf</code> and <code>scanf</code> are from stdio.h.</p>
<p data-sourcepos="53:1-53:25" dir="auto"><strong>What is <code>&lt;conio.h&gt;</code> ?</strong></p>
<p data-sourcepos="55:1-55:218" dir="auto">It stands for 'Console Input Output'. conio.h is a header file which is mostly used by MS-DOS compilers like TurboC. Functions like <code>getch()</code> are from conio.h and <em>won't work unless conio.h is included in the program</em>.</p>
<p data-sourcepos="57:1-57:27" dir="auto"><strong>What is <code>void main()</code> ?</strong></p>
<p data-sourcepos="59:1-59:113" dir="auto">void main() is the entry point for execution in C program. It is where the main code of your program is written.</p>
<blockquote data-sourcepos="60:1-60:106">
<p data-sourcepos="60:3-60:106" dir="auto">Main function is the heart of C program. Everything inside the main block will be code that is executed.</p>
</blockquote>
<p data-sourcepos="62:1-62:24" dir="auto"><strong>What is <code>printf()</code> ?</strong></p>
<p data-sourcepos="64:1-64:103" dir="auto">printf() is a function used to output text to the screen. In our example it will output "Hello World!".</p>
<p data-sourcepos="66:1-66:23" dir="auto"><strong>What is <code>getch()</code> ?</strong></p>
