## Peptone Agar

|Compound| Amount | Unit |
|:-------|-------:|-----:|
|Peptone|10.0|g|
|NaCl|5.0|g|
|Agar|2.0|g|
|Demi water| up to 1000|mL|

Set pH ~7.2 

### Biohacker Alternative

* Please note: peptone is very dusty, so try not to inhale it.
* Peptone can be found on eBay.
